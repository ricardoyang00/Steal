# Environment variables
## Google OAuth
- **GOOGLE_CLIENT_ID:** Your Google OAuth client ID.
- **GOOGLE_CLIENT_SECRET:** Your Google OAuth client secret.
- **GOOGLE_CALL_BACK_ROUTE:** The callback URL for Google OAuth, typically http://127.0.0.1:8000/auth/google/call-back.


## Email Configuration
- **MAIL_MAILER:** The mailer to use, typically smtp.
- **MAIL_HOST:** The SMTP server host, e.g., sandbox.smtp.- **mailtrap.io.
- **MAIL_PORT:** The SMTP server port.
- **MAIL_USERNAME:** The SMTP server username.
- **MAIL_PASSWORD:** The SMTP server password.
- **MAIL_ENCRYPTION:** The encryption method, typically tls.
- **MAIL_FROM_ADDRESS:** The email address from which emails will be sent.
- **MAIL_FROM_NAME:** The name that will appear on sent emails.
